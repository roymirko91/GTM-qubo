## 🎓 Aprende Estos Frameworks

Estos 6 son los que deberías dominar:

1. **MEDDPICC** - Para deals enterprise
   - Metrics / Economic Buyer / Decision Criteria / Decision Process / Pain / Impending Event / Champion

2. **SPIN Selling** - Para consultativo
   - Situation / Problem / Implication / Need-Payoff

3. **Challenger Sale** - Para posicionarse
   - Teach / Tailor / Take Control

4. **Sandler** - Para calificar rápido
   - Pain Funnel / Budget / Decision

5. **Value Selling** - Para ROI
   - Current State → Future State → Gap → Solution → Value

6. **Gap Selling** - Para buyer journey
   - Identify Gap → Explore Gap → Prioritize Gap → Solve Gap

Úsalos consistentemente y verás resultados.
